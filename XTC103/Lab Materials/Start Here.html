Before you begin

This is a Xamarin Test Cloud class which supports both Xamarin.iOS and Xamarin.Android. To complete the coding exercises, you will need a Xamarin development environment with a local device (not a simulator) to run the exercise on. In addition, you will need to sign up for a Test Cloud account - you can get a coupon for 25 free hours from your Profile page.

Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This class is a continuation of the XTC102 - Working with Xamarin.UITest class and shows how to run acceptance tests on your local devices and then how to push those tests up to the cloud with Xamarin Test Cloud.

Objectives

  1. Testing on physical devices
  2. How to deploy to Test Cloud

Lecture

XTC103 - Deploying to Xamarin Test Cloud.

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Deploying Xamarin.UITests to a local device
    Exercise
    In this exercise, we will take a set of UI Tests and run them against an application installed on a local device.
  2. Uploading a project to Test Cloud
    Exercise
    In this demonstration, you will see how to upload your tests to Xamarin Test Cloud, execute them and then analyze the results in the Test Cloud portal
